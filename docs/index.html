llm-arch-reviewer

interactive architecture diagrams for LLM inference, with profile data overlaid
GitHub ↗ how it works add a model

One modular, zoomable, click-to-drill architecture diagram per model, with measured per-block ms and kernel breakdown overlaid from real trace data (sglang prefill / decode, with PDF projections where available).

Everything is data-driven: nodes, edges, shapes, drill relationships — all described in IR YAML; the renderer (ELK + SVG) is fully model-agnostic. Adding a new model means dropping a new models/<name>/ directory; no JS changes required.

about

this site is a hand-authored layout combined with auto-generated profile overlay. see the repo for IR schema, build pipeline, and how to add a new model.

profile data uses sglang traces (prefill / decode) post-processed with kernel-name classification; numbers are rolled up per architectural block, with both per-stage and per-block percentages exposed in the detail panel.
